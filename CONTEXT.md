# CONTEXT

Ημερομηνία: 2025-11-01

Στόχος αυτού του αρχείου είναι να αποτελεί «ζωντανό» σύντομο υπόμνημα για γρήγορη προσανατολισμό και για να μπορώ να βοηθώ άμεσα σε επόμενες επεμβάσεις.

## Επισκόπηση έργου

- Όνομα/τίτλος: Νομοί & Πρωτεύουσες Μακεδονίας – Παιχνίδι μνήμης + Διαδραστικός χάρτης
- Περιγραφή (1–2 προτάσεις): Στατικός ιστότοπος με διαδραστικό χάρτη (Leaflet + OpenStreetMap) και παιχνίδι μνήμης δύο πινάκων για τα 13 ζεύγη Νομών/Πρωτευουσών της Μακεδονίας. Τα σημεία στο χάρτη αντλούνται από Wikidata (SPARQL) με ασφαλές fallback σε τοπικές συντεταγμένες.
- Κατάσταση: Αρχική μονοσέλιδη υλοποίηση στο `index.html`.

## Κοινό-στόχος & βασικές ροές

- Κοινό: Μαθητές/εκπαιδευτικοί, ελληνόφωνοι χρήστες.
- Ροές:
  1. Μελέτη των πρωτευουσών στον χάρτη.
  2. Παιχνίδι μνήμης δύο πινάκων: Νομοί ↔ Πρωτεύουσες.
  3. Λειτουργία «Μελέτης» που γυρίζει όλες τις κάρτες.

## Τεχνολογίες

- HTML/CSS/JS μόνο (χωρίς build tool).
- Leaflet 1.9.4 από CDN, OpenStreetMap πλακίδια.
- Wikidata SPARQL endpoint για συντεταγμένες (προαιρετικά, με graceful fallback).
- Γλώσσα περιεχομένου: el (Greek).

## Design System (first pass)

- Χρωματική παλέτα σε CSS variables στο `index.html` (`:root`).
- Τυπογραφία: system-ui stack.
- Συστατικά: κάρτες (flip), panels, HUD, κουμπιά.

## Δεδομένα / APIs

- Στατικά δεδομένα 13 ζευγών στο script (πίνακας `NOMOI_MAKEDONIAS`).
- SPARQL call στο Wikidata για συντεταγμένες, με fallback στο `CAPITAL_COORDS_FALLBACK`.
- Χωρίς auth/secrets.
