# Νομοί & Πρωτεύουσες Μακεδονίας – Παιχνίδι μνήμης + Διαδραστικός χάρτης

Στατικός ιστότοπος με Leaflet χάρτη και παιχνίδι μνήμης δύο πινάκων για τα 13 ζεύγη Νομών/Πρωτευουσών της Μακεδονίας. Τα σημεία στο χάρτη φορτώνονται από Wikidata (SPARQL) με ασφαλές fallback.

English summary: Static single-page app (Leaflet map + two-board memory game) for 13 Macedonia prefectures and their capitals. Wikidata SPARQL for coordinates with local fallback.

## Δομή

- `index.html` — όλη η εφαρμογή (HTML/CSS/JS inlined)
- `CONTEXT.md` — ζωντανό υπόμνημα έργου (overview, ροές, τεχνολογίες, κ.λπ.)
- `TODO.md` — προτεραιοποιημένα tasks (Now/Next/Later)
- `.vscode/` — ρυθμίσεις editor & προτεινόμενα extensions
- `.editorconfig` — κοινές ρυθμίσεις μορφοποίησης

## Γρήγορη εκτέλεση

Προτείνεται το VS Code Live Server (υπάρχει ως recommendation):

1. Άνοιξε τον φάκελο στο VS Code.
2. Κάνε δεξί κλικ στο `index.html` → “Open with Live Server”.

Εναλλακτικά, μπορείς να ανοίξεις απευθείας το αρχείο στον browser (ο χάρτης θα λειτουργήσει κανονικά, αλλά προτιμάται τοπικός server για μελλοντικές επεκτάσεις).

## Σημειώσεις

- Αν αποτύχει η φόρτωση του Leaflet από CDN, η εφαρμογή συνεχίζει να λειτουργεί χωρίς χάρτη.
- Η γλώσσα περιεχομένου είναι ελληνικά (`<html lang="el">`).

## Εμφάνιση: Σταθερό 16:9 & προσανατολισμός

- Η εφαρμογή τρέχει μέσα σε «σκηνή» 16:9 (1280×720) που κλιμακώνεται ομοιόμορφα σε κάθε οθόνη με letterboxing όπου χρειάζεται. Έτσι, η διάταξη/αναλογίες είναι ακριβώς ίδιες σε κάθε πλατφόρμα.
- Σε κινητά/τάμπλετ επιβάλλεται οριζόντιος προσανατολισμός. Αν η συσκευή είναι σε κάθετο, εμφανίζεται overlay που ζητά περιστροφή. Όπου υποστηρίζεται, γίνεται προσπάθεια κλειδώματος σε landscape με το Screen Orientation API πάνω σε χρήσιμη ενέργεια (π.χ. «Νέα παρτίδα»).
- Απενεργοποιείται το pinch-zoom (viewport `user-scalable=no`) ώστε η κλίμακα να ελέγχεται αποκλειστικά από την εφαρμογή.
